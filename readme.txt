Project info:
php version = 7.3.x
npm version = 6.14.12
node version = 14.16.1

Code style:
Табуляція (4 знака)

Звичайні коментарі:
{# коментар #}

Коментарі з правками:
{# !Доробити логіку #}
...Логіка...
{# !=============== #}

PS:
    Всі змінні стилів знаходяться в файлі ./assets/scss/varibales.scss
    Всі класи для тексту знаходяться в файлі ./assets/scss/texts.scss
    Всі міксини знаходяться в файлі ./assets/scss/mixins.scss
    Всі підключені шрифти знаходяться в файлі ./assets/scss/fonts.scss
    Також є файл normalize.scss, який містить стилі для нуліфікації зайвих стилів
    Також є файл global.scss, який містить загальні підключені стилі та базові стилі для проекту
